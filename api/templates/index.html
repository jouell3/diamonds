You have access to two different API: - Single prediction - Batch prediction

Single prediction

To predict the price of a single diamond, you can use the following endpoint:

POST /predict

With the following JSON body:

{
    "carat": float,
    "cut": "string",
    "color": "string",
    "clarity": "string",
    "depth": float,
    "table": float,
    "x": float,
    "y": float,
    "z": float
}

Batch prediction

To predict the price of multiple diamonds at once, you can use the following endpoint:

POST /predict_batch

With the following JSON body:

[
    {
        "carat": float,
        "cut": "string",
        "color": "string",
        "clarity": "string",
        "depth": float,
        "table": float,
        "x": float,
        "y": float,
        "z": float
    }
]
